使用Codex辅助给Clash For Windows换内核
发布于：2026-05-29 16:09:26 访问：

前提条件

我一直使用的是Clash Verge，但是我这个人比较怀念前面的Clash for windows版本，但是核心等等都已经全部都落后了，但是没有关系，直接换内核就好了

下载Clash For Windows

因为官方删除仓库的原因所以我打算去archive.org上下载

下载完成之后进行按照到c盘(建议)

找到Clash For Windows内核

以下是举例内核位置，内核名称为 clash-win64 （如果是32位系统就是32）

您可以在桌面右键 Clash for Windows图标 点击 打开文件所在的位置 。
您就会跳到Clash for Windows文件的目录，然后继续到 \resources\static\files\win\x64
您会看到 clash-win64的名称

开始工作

安装mihomo内核

找到最新的的版本，下载mihomo-windows-amd64-xxxx.zip

注意:xxxx是每次都不是固定的，所以要自己去判断，只要找到amd64就没有问题了

下载下来之后放到跟Clash For Windows内核的目录

Codex安装

进入内核目录打开Codex

输入提示词
安装内核提示词

这一步建议直接安装TUN虚拟网卡模式,如果不会，可以不用安装

安装内核跟TUN提示词

注意:上面两种提示词只需要选择一种即可
